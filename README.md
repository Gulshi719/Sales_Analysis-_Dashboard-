# Sales Analysis Dashboard

## Project Overview
Interactive Sales Analysis Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and KPI Cards.

## KPIs
- Total Sales: ₹2.30M
- Total Profit: ₹286.4K
- Total Customers: 793
- Profit Margin: 12.47%
- Top Region: West

## Features
- Region-wise Sales & Profit Analysis
- Category-wise Sales & Profit Analysis
- Monthly Sales Trend
- Monthly Profit Trend
- Top 5 Products Analysis
- Interactive Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
